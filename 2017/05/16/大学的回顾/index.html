大学的回顾

发表于 2017-05-16 | 分类于 自己


在华农四年行将结束，似乎该写点什么来纪念这段大学时光。

缘起

华南农业大学是一所普通的一本大学，在大一时我曾听说华农是211工程级别的一本

华南农业大学是全国重点大学,广东省和农业部共建的“211工程”大学,广东省高水平大学重点建设高校。

略有点小兴奋。后来才知道省部共建的211和国家级别的211根本就不是一回事。ORZ，总有种被骗的感觉！:-)

说起与华农的缘分不得不提这样一张相片

这张图是拍摄于2005年的5月份，是我表哥（左边的小个子）的毕业照。当时我才是一名小学生。我就穿上学士服，顺利从华农“毕业”了233.所以我可以自豪地说我这大学四年其实是来华农“复读”的。右边的也是我表哥，不过他是华工土木系毕业的。他们哥俩一个是华工的一个是华农的简直太有喜感了。

初来乍到

初次来到华农就觉得这里分外清爽。宽阔的校园，茂密的植被让人心旷神怡。

最令我震惊的是这里的牛神兽居然能自由跑动。好一副自然和谐的奇观。

我的宿舍在华农的最东边，是个被称为启林南的地方。基本上已经不属于华农的传统校区了。是华农新占的地方。“启林”区的解释也很有意思

启林，启林，以启山林。

一方面说明了前辈们在这里开荒的日子的艰苦。还有虽然有点煞风景，我还是得说一句，启林区的饭堂荷园巨难吃！

从我们宿舍望出去能看到天河区的一些老旧建筑，还有远处新广州市的繁华的天际线。风景是很美的。

不过宿舍里就显得有点破烂了

在大一的时候，各种奇奇怪怪的联谊活动十分频繁。主要就是在夜间的操场里，两个班的男女玩莫可名状的游戏。还有各个社团里的社交活动。升上大学之前我本下定决心在大学一定要广交朋友，做一个开朗的人。。。但经过各种高密度的社交场合下来，我心里面只有一个字那就“累”.
我慢慢认清自己的喜好。我退了，大量我不感兴趣的部门，只留下一个

那就是

广播台

这是一个校级的组织光是面试就足足有4轮。因为我进的是编辑部，所以还得额外提交稿件。

虽说是校级组织但总部却在主校区的某个上世纪风格的老房子里。有种微妙的复古的违和感。

广播台每天中午下午都会在全校播放广播节目，每天两档。主要分工是记者，编辑，播音，技术。我作为编辑主要协作对象就是播音，需要在写好稿子的基础上，与播音一起预稿练习。

在这里认识了朋友。

在这里收获了荣誉

最重要的是自己的创作能够在全校范围内播出真的是很酷的一件事好吗！

回忆

说起在大学的日常，主要就是看书

那时候的我通过大量高密度的社交发现，自己还是喜欢一对一的社交还有独处。自然就开始沉迷起看书来。大一大二几乎把图书馆里各个科目的书都看了一遍。科幻小说，心理学，人类学，计算机等

在看书之余写稿。

然后和朋友们一起出去耍。

去游乐园

去爬山

玩枪战

逗猫

尾声

华农真的很美，在这四年过得很有意义。

容我对你说声

“再见！华农！”

虽然伤感，但是不必流泪只要保持微笑就好。
